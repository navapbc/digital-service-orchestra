# Add monitoring.tool_errors=true to workflow-config.conf

Add monitoring.tool_errors=true under a # Monitoring section in workflow-config.conf at the repo root. Create the section if it does not exist. This enables full tool error tracking for the DSO project itself.

Other projects that do not have this key are unaffected (absent = disabled by default).
