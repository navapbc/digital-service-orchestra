# Bug: fix-bug never triggers COMPLEX_ESCALATION — all bugs resolved by solo sub-agents without multi-agent planning

During /dso:debug-everything sessions, all bugs are dispatched to individual fix-bug sub-agents. None are ever escalated to COMPLEX (Step 4.5 of fix-bug), which would trigger orchestrator-level re-dispatch with multi-agent planning. This means the COMPLEX_ESCALATION path in Phase 6 Step 3a of debug-everything is dead code — it never fires in practice. Either the complexity-evaluator threshold is too high (never classifies bugs as COMPLEX), or the fix-bug scoring rubric doesn't produce scores high enough to trigger escalation. The result is that bugs requiring cross-system fixes or parallel agent coordination are attempted by solo agents who may produce incomplete or incorrect fixes.
