# Implement ticket create (UUID gen, CREATE event, atomic write via ticket-lib.sh)


## Description

Implement `plugins/dso/scripts/ticket-create.sh` — the create subcommand.

### Usage
```
ticket create <ticket_type> <title> [parent_id]
```
Allowed ticket types: bug, epic, story, task

### Implementation steps:
1. Validate `ticket_type` is one of: bug, epic, story, task; exit 1 with usage message if invalid
2. Validate `title` is non-empty; exit 1 with usage message if empty
3. Generate a collision-resistant short ID:
   - Use `python3 -c "import uuid; u=str(uuid.uuid4()).replace('-',''); print(u[:4]+'-'+u[4:8])"` for short ID (format: `xxxx-xxxx`)
   - Generate a full UUID4 for the event's `uuid` field
4. Read env_id from `.tickets-tracker/.env-id` (fail if not found — means init was not run)
5. Get timestamp: `python3 -c "import time; print(int(time.time()))"`
6. Get author: `git config user.name`
7. Construct CREATE event JSON using **python3 json.dumps** (never bash heredoc):
   ```python
   import json, sys
   event = {
     "timestamp": <int>,
     "uuid": "<uuid4>",
     "event_type": "CREATE",
     "env_id": "<env_id>",
     "author": "<author>",
     "data": {
       "ticket_type": "<type>",
       "title": "<title>",
       "parent_id": None  # or string if provided
     }
   }
   print(json.dumps(event, ensure_ascii=False))
   ```
8. Write to a temp file in `.tickets-tracker/` (same filesystem as target for atomic rename)
9. Call `write_commit_event <ticket_id> <temp_file>` from `ticket-lib.sh`
10. Print `<ticket_id>` to stdout (only the ID — no other output)

### Filename convention (from contract w21-mtvm):
`.tickets-tracker/<ticket_id>/<timestamp>-<uuid>-CREATE.json`
Note: `<uuid>` in filename uses the full UUID4 with hyphens.

### Constraints:
- Source `ticket-lib.sh` at top of script
- All JSON via Python json.dumps — explicit `encoding='utf-8'` for file writes
- ID format: `<4hex>-<4hex>` (e.g., `a1b2-c3d4`)
- Parent ID is validated to exist in `.tickets-tracker/` if provided (CREATE event check)
