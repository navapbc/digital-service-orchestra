# Create a setup skill

We should create a new skill that walks the user through setup for a client project. The skill should include interactively customizing and copying dso templates to the project, interactively creating a dso configuration file with appropriate settings for the project (we should suggest defaults based on the project), configuring our git hooks for key workflow pieces like our review gate and formatting, etc.
