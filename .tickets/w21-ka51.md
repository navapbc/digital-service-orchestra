## Detection category: database presence (key=db_present, db_services)

Check for indicators of database usage:
1. docker-compose.yml or docker-compose.yaml: grep for image: postgres|mysql|mariadb|mongodb|redis
2. Dockerfile: grep for ENV.*DATABASE_URL or EXPOSE.*5432|3306|27017
3. Config files: grep for DATABASE_URL, db_url, db.url patterns in .env, .env.example, config/*.{py,yml,yaml,rb}

```bash
db_present="false"
db_services=""
for compose_file in docker-compose.yml docker-compose.yaml; do
  if [[ -f "$PROJECT_DIR/$compose_file" ]]; then
    matched=$(grep -oiE 'image: *(postgres|mysql|mariadb|mongodb|redis)' "$PROJECT_DIR/$compose_file" | grep -oiE 'postgres|mysql|mariadb|mongodb|redis' | sort -u | tr '\n' ',' | sed 's/,$//')
    if [[ -n "$matched" ]]; then db_present="true"; db_services+="$matched,"; fi
  fi
done
db_services="${db_services%,}"
echo "db_present=$db_present"
echo "db_services=$db_services"
```

## Detection category: port numbers (key=ports)

Detect port numbers from workflow-config.conf or common config files:
1. workflow-config.conf: grep for port= or ports= key
2. docker-compose.yml: grep for 'ports:' section patterns like '- "8080:8080"'

```bash
ports=""
if [[ -f "$PROJECT_DIR/workflow-config.conf" ]]; then
  conf_port=$(grep -m1 '^port=' "$PROJECT_DIR/workflow-config.conf" | cut -d= -f2-)
  [[ -n "$conf_port" ]] && ports+="$conf_port,"
fi
# docker-compose port extraction
for compose_file in docker-compose.yml docker-compose.yaml; do
  if [[ -f "$PROJECT_DIR/$compose_file" ]]; then
    compose_ports=$(grep -oE '"[0-9]+:[0-9]+"' "$PROJECT_DIR/$compose_file" | grep -oE '^"[0-9]+' | tr -d '"' | sort -u | tr '\n' ',' | sed 's/,$//')
    [[ -n "$compose_ports" ]] && ports+="$compose_ports,"
  fi
done
ports="${ports%,}"
echo "ports=$ports"
```

## Detection category: version file candidates (key=version_files)

Detect files that likely contain the project's semantic version string:
1. package.json: check for 'version' key at top level
2. pyproject.toml: check for 'version =' in [project] or [tool.poetry] section
3. Cargo.toml: check for 'version =' in [package] section

```bash
version_files=""
if [[ -f "$PROJECT_DIR/package.json" ]]; then
  python3 -c "import json,sys; d=json.load(open(sys.argv[1])); sys.exit(0 if 'version' in d else 1)" "$PROJECT_DIR/package.json" 2>/dev/null && version_files+="package.json,"
fi
if [[ -f "$PROJECT_DIR/pyproject.toml" ]]; then
  grep -qE '^version =' "$PROJECT_DIR/pyproject.toml" && version_files+="pyproject.toml,"
fi
if [[ -f "$PROJECT_DIR/Cargo.toml" ]]; then
  grep -qE '^version =' "$PROJECT_DIR/Cargo.toml" && version_files+="Cargo.toml,"
fi
version_files="${version_files%,}"
echo "version_files=$version_files"
```
