2. Stack detection (delegates to detect-stack.sh: python-poetry, node-npm, golang, rust-cargo, convention-based, unknown)

6. Python version (pyproject.toml requires-python, .python-version file, python3 binary fallback)
7. Installed CLI dependencies (acli, PyYAML, pre-commit, shasum: present/absent with confidence=high)
8. Existing file presence (CLAUDE.md, KNOWN-ISSUES.md, .pre-commit-config.yaml, workflow-config.conf)
9. Port numbers from workflow-config.conf
10. Version file candidates (package.json version field, pyproject.toml version field)
11. Confidence degradation: when detection heuristic is uncertain, output includes confidence=low
12. Graceful degradation: script exits 0 when optional files/tools are absent
