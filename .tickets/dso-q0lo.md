# Prevent the explosion of documentation created by LLMs

LLM agents frequently create and expand documentation beyond what is useful for the project. We want to ensure that documentation is updated to accurately reflect the project and key information is stored for future reference without creating an ever-growing number and size of documents for future agents to sift through
