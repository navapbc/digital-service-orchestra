# IMPL-TARGETS-PYVER-DEPS: Implement target enumeration, Python version, and installed CLI dependency detection

## Implementation — Three detection categories

Add three detection categories to plugins/dso/scripts/project-detect.sh.

## Detection category: target enumeration (key=targets)

Detect available build targets from the project's build file:

### Makefile targets
Grep for lines matching '^<target>:' pattern. Report targets: test, lint, format, format-check, build, clean, install
```bash
if [[ -f "$PROJECT_DIR/Makefile" ]]; then
  makefile_targets=""
  for t in test lint format format-check build clean install; do
    if grep -q "^${t}:" "$PROJECT_DIR/Makefile"; then
      makefile_targets+="$t,"
    fi
  done
  echo "targets=${makefile_targets%,}"
fi
```

### package.json scripts
Use python3 to parse package.json and emit the scripts keys:
```bash
if [[ -f "$PROJECT_DIR/package.json" ]]; then
  pkg_scripts=$(python3 -c "
import json, sys
with open(sys.argv[1]) as f:
    d = json.load(f)
print(','.join(d.get('scripts', {}).keys()))
" "$PROJECT_DIR/package.json" 2>/dev/null || echo "")
  echo "targets=$pkg_scripts"
fi
```

The Makefile path takes priority if both exist (aligns with detect-stack.sh priority). If no build file has recognized targets, emit targets= (empty).

## Detection category: Python version (key=python_version, python_version_confidence)

Priority order (first match wins, confidence=high unless noted):
1. pyproject.toml requires-python field (e.g., '>=3.11'): extract version number, confidence=high
2. .python-version file (single line): read directly, confidence=high
3. python3 binary: run 'python3 --version 2>/dev/null', parse output, confidence=low (binary may not match project intent)
4. Fallback: python_version=unknown, python_version_confidence=low

```bash
python_version="unknown"
python_version_confidence="low"
if [[ -f "$PROJECT_DIR/pyproject.toml" ]]; then
  ver=$(grep -m1 'requires-python' "$PROJECT_DIR/pyproject.toml" | grep -oE '[0-9]+\.[0-9]+' | head -1)
  if [[ -n "$ver" ]]; then python_version="$ver"; python_version_confidence="high"; fi
fi
# ... fallthrough to .python-version then python3 binary
echo "python_version=$python_version"
echo "python_version_confidence=$python_version_confidence"
```

## Detection category: installed CLI dependencies (key=installed_deps)

Check whether each optional dependency is installed using 'command -v':
- acli
- pre-commit
- shasum

Check PyYAML via python3 -c 'import yaml' (exits 0 if available).

Emit a comma-separated list of installed dependency names. Also emit individual boolean keys:
- acli_installed=true|false
- pre_commit_installed=true|false
- shasum_installed=true|false
- pyyaml_installed=true|false
```bash
installed_deps=""
command -v acli >/dev/null 2>&1 && installed_deps+="acli," && echo "acli_installed=true" || echo "acli_installed=false"
# ... similarly for pre-commit, shasum
python3 -c 'import yaml' >/dev/null 2>&1 && installed_deps+="pyyaml," && echo "pyyaml_installed=true" || echo "pyyaml_installed=false"
echo "installed_deps=${installed_deps%,}"
```
