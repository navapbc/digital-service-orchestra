# Implement auto-init guard in ticket dispatcher (transparent first-use initialization)


## Description

Update `plugins/dso/scripts/ticket` dispatcher to silently run `ticket init` if `.tickets-tracker/` does not exist before routing any subcommand.

This satisfies Done Definition 6: "every `ticket` command auto-initializes if `.tickets-tracker/` does not exist — `ticket init` runs silently on first use."

### Implementation steps:
1. In `plugins/dso/scripts/ticket` dispatcher, add a guard near the top (after argument parsing, before subcommand routing):
   ```bash
   _ensure_initialized() {
     if [[ ! -d ".tickets-tracker" ]]; then
       # Silent auto-init (no stdout output from init on auto-init)
       bash "$(dirname "$0")/ticket-init.sh" --silent 2>/dev/null || {
         echo "Error: ticket system initialization failed. Run 'ticket init' manually." >&2
         exit 1
       }
     fi
   }
   ```
2. Add `--silent` flag to `ticket-init.sh` that suppresses the "Ticket system initialized." success message (so `ticket create task "foo"` auto-inits silently and only prints the ticket ID)
3. Call `_ensure_initialized` before routing to create/show (NOT before `ticket init` itself — avoid recursion)
4. `ticket init` when called explicitly still prints the success message (no --silent flag passed)

### RED test to write (in `tests/scripts/test-ticket-autoinit.sh`):
1. `test_ticket_autoinit_on_create` — assert that `ticket create task "foo"` in a fresh repo (no prior `ticket init`) succeeds (exit 0) and creates `.tickets-tracker/` automatically
2. `test_ticket_autoinit_on_show` — assert that `ticket show` in a fresh repo (no prior init) exits non-zero gracefully (init succeeds, but no ticket to show) without crashing with "tickets-tracker not found"
3. `test_ticket_explicit_init_still_prints_message` — assert `ticket init` (explicit, not auto) prints "Ticket system initialized." to stdout

Write these tests first (RED), then implement (GREEN).

Depends on: w21-1plz (init implementation), w21-g3x6 (create implementation), w21-ymip (show implementation)
